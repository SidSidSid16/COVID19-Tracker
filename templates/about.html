The COVID-19 Dataset That I Am Using
I am using a dataset provided and maintained by Microsoft Bing. You can access it by clicking here. This dataset consists of around 500k rows, more rows (like hundreds of them) are added everyday. For the most part, it's alright to use. However, there are so many inconsistencies. Why didn't I use an API? Well, all of them are paid. The free ones have ludicrous rate-limiting. In fairness, this project is extremely small scale, I doubt I'll get any more than 5 unique visitors per month, so the rate-limited APIs would've worked great. But, I wanted to try out Pandas and see how it and Python works with enormous datasets such as this.

Before I forget, there is a license that Bing has provided that I need to comply with. As per this license I need to add a link to the Bing COVID-19 dashboard, so, here it is.

A huge let down, as I was expecting this dataset to hold an open-source license, instead it holds a really controlling one. It really is sad to see big companies such as Microsoft take advantage of their size to hurt smaller companies and people. The quality of this dataset is quite debatable too, a huge about of inconsistencies, to put it simply, a Microsoft employee probably built a simple web scraper which dumps numbers into a .csv file. The greed for money is a vicious circle I guess.

But it doesn't matter. This program runs fairly well I suppose.

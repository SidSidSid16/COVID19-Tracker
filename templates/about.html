This is a Python web application which uses a large dataset consisting of COVID-19 data from many countries and regions. The aims of this projects are to pratise Python and learn to use the Flask and Pandas libraries.

The code may not be the best quality as this is my first every fully-fledged python project. You can help me improve this code. Click here to contact me. Click here to go to the repository where the source code is hosted
The COVID-19 Dataset That I Am Using
I am using a dataset provided and maintained by Microsoft Bing. You can access it by clicking here. This dataset consists of around 500k rows, more rows (like hundreds of them) are added everyday. For the most part, it's alright to use. However, there are many inconsistencies. The reason why I decided not to use an API was because I wanted to try out and learn Pandas and see how it and Python performs with enormous datasets such as this.
